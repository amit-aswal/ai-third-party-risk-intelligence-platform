# Practical Portfolio Roadmap

## Project Title

AI-Powered Third-Party Risk, Contract Renewal & Spend Leakage Intelligence Platform

## Portfolio Goal

Build a real-data-driven analytics and AI project using public datasets and APIs. The project should solve practical business problems around vendor risk, contract risk, spend risk, news/adverse-media risk, compliance review, and financial risk.

## Final Phases

1. Real Dataset Access & Validation
2. Contract Intelligence using CUAD
3. Vendor Spend Analytics using USAspending
4. Vendor Entity Matching
5. News Risk Monitoring using GDELT
6. SEC + Compliance Risk
7. Final Vendor Risk Scoring Engine
8. Power BI Dashboard
9. Streamlit App
10. Automation, A/B Testing & GitHub Polish

## First Build Step

Start with Phase 1 by validating that CUAD, USAspending, GDELT, SEC EDGAR, and OpenSanctions sample access work locally.
